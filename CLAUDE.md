# CLAUDE.md

Single static HTML site (`index.html` only) deployed to izborenmonitor.com via GitHub Pages. No build step — push to `main` deploys to production.

- Test locally: open `index.html` in a browser
- Small fixes: push directly to `main`; bigger changes: open a PR
- The map is one of multiple planned projects under this site; the site's identity is fair and transparent elections broadly — not just the map
- Map iframe comes from `karta.izborenmonitor.com` (separate project)
- Bilingual: Bulgarian primary, English secondary
- Flag colors: white `#fff`, green `#00966E`, red `#D62612`
